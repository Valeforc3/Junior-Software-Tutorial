# Components Directory

This directory holds reusable React components for the application.

## Files
- `ContactForm.jsx`: A form for user contact submissions.
- `Footer.jsx`: Footer component with links and copyright.
- `Header.jsx`: Header component for navigation.
    > The header and footer files are referenced in "./app/layout.jsx", this is to ensure standard page layout across every page.

## Purpose
Components promote code reusability. Use these in pages or layouts to build consistent UI.

# Data Directory

This directory contains static data files used throughout the app.

## Files
- `tasks.jsx`: Exports an array of tasks for the tutorial project.

## Purpose
Centralizes data for easy management. In a real app, these files could connect to a database or API to allow the data to be updated live across multiple locations.

> Although it feels complicated, it's easier to update your data from a central location, and reference it on other pages. Implementing structure like this means you don't have to change each page individually.

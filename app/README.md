# App Directory

This directory contains the Next.js application using the App Router.

## Structure
- `layout.jsx`: The Root layout for the entire app.
- `page.jsx`: The Home page.
    > Remember, every page you see in a React App is a folder with a "page.jsx" file within it.
- `components/`: Shared React components.
- `data/`: Data files used by the app.
    > Other folders within the "app" folder are specific pages and can be customised however your page needs to be layed out, just remember, to keep the directory structure flat.

## Purpose
The App Router is Next.js's modern routing system. Files here define routes and layouts for your web app.

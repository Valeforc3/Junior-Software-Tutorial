### Getting started with this project

To get started with this project, you will need the following:
1. A GitHub Account (Explained in the next article)
2. Visual Studio Code installed [Installer found here](https://code.visualstudio.com/download)
    - In addition to this, you also need the following extensions (trust me, it will make life easier), you can browse for extensions in the [Visual Studio Marketplace](https://marketplace.visualstudio.com/vscode)
    1. [Remote Repositories](https://marketplace.visualstudio.com/items?itemName=ms-vscode.remote-repositories)
    2. [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
3. An account with [Vercel](https://vercel.com/) to deploy the project for free


In addition to this, it's a good idea to understand how "markdown files" work, that's the format that all these Wiki Files are written.
> You can tell it's a markdown file because it has the .md extension. [Here is the formatting page I use](https://www.markdownguide.org/basic-syntax/#links)
